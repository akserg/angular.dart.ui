Angular UI [![Build Status](https://travis-ci.org/akserg/angular.dart.ui.svg?branch=master)](https://travis-ci.org/akserg/angular.dart.ui)
===============

Port of Angular-UI to Dart.

##Bootstrap directives and components

- Checkbox and RadioButton
- DropdownToggle
- Collapse
- Alert
- ProgressBar
- Modal
- Accordion
- Rating
- Datepicker (partially implemented)
- Tabs
- Drag and Drop support
- Carousel
- Timepicker
- Pagination
- Tooltip
- Popover
- Typehead

*Note: Drag and Drop support is experimental feature and API can be changed at any time in the future.*
